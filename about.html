About this project — यस परियोजनाको बारेमा

A modern, searchable rebuild of the Federal Parliament Secretariat's printed telephone directory.

What this is

The Federal Parliament Secretariat publishes a printed Telephone Directory (2083 BS edition, 106 pages) listing every Member of the House of Representatives and National Assembly, along with Secretariat leadership and offices. It's comprehensive but hard to use: no search, dense tables, and not mobile-friendly.

This project extracts that same information into a searchable website, spreadsheet, and structured data files — while preserving every contact record from the source.

How the data was built

The source PDF is typeset in Preeti, a legacy pre-Unicode Nepali font from the 1990s. Fonts like this render correctly on screen but store text internally in a way that standard PDF text extraction reads back as corrupted characters — copying text from the PDF produces garbage, even though it looks correct when printed or viewed.

To recover the data accurately, each of the 106 pages was rendered as a high-resolution image and read with OCR (optical character recognition), which reads the visual glyphs rather than the broken underlying text layer. Phone numbers and email addresses — which are typed in Latin script — were cross-validated directly against the PDF's native text layer, which extracts them correctly. This gives two independent sources of truth for the fields that matter most.

Data confidence

Every record carries a data_confidence field:

ValueMeaning
highRow matched cleanly across both extraction methods.
medium_sequential / medium_fallbackRecovered via a secondary matching pass — spot-check advised.
low_needs_review / low_no_chunkRecommend manual verification against the source PDF page (listed in each record).

Mobile numbers and email addresses are extracted from the PDF's native text layer, not OCR, so these fields should be faithful to the source regardless of confidence level. District/Constituency and Political Party fields were split algorithmically from OCR'd text and are the fields most likely to need a second look.

What's included / what's next

Included: all 275 House of Representatives members and 57 National Assembly members, with name, house, district/constituency, political party, mobile number(s), and email where available. A supplementary raw extract of Secretariat leadership and office contacts is also included in the downloads.

Not yet included: the full Secretariat staff directory, province/district/ministry directories, and other public-service listings (hospitals, embassies, banks, media) that also appear in the source publication.

Corrections

If you spot an incorrect number, email, or spelling, please open an issue on the project's GitHub repository with the person's name and the correct information.
